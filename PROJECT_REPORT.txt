RUNNING THE PROJECT
================================================================================

Quick Start:
  1. pip install -r requirements.txt
  2. python run_all_tests.py
  3. python cli.py csv-to-json fixtures_sample.csv

Common Commands:
  # Convert CSV to JSON
  python cli.py csv-to-json data.csv -o output.json

  # Batch process
  python cli.py csv-to-json "*.csv" --batch -o json/

  # Convert Markdown
  python cli.py markdown-to-html README.md -o out.html

  # Use piping
  cat data.csv | python cli.py csv-to-json

Run Tests:
  python run_all_tests.py          # All 74 tests
  pytest test_csv_json.py -v       # Specific suite
  pytest -k "roundtrip" -v         # Match pattern
